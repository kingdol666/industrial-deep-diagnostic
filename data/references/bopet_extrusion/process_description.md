# BOPET 双向拉伸聚酯薄膜 — 纵向拉伸（MD）工艺描述

## 系统概述

本产线为 BOPET（Biaxially Oriented Polyethylene Terephthalate）薄膜生产线。PET 树脂经挤出机熔融挤出后，通过铸片辊冷却成型，再进入纵向拉伸机（MDO, Machine Direction Orientation）进行单向拉伸。本次诊断聚焦于 MD 拉伸段及挤出段的工艺参数与薄膜表面"划伤"缺陷之间的关联分析。

## 工艺流程

```
PET切片 → 干燥 → 挤出机熔融 → T型模头 → 铸片辊冷却 → 纵向拉伸(MD) → 横向拉伸(TD) → 热定型 → 收卷
                                                      ↑
                                                 本分析覆盖区域
```

## 划伤缺陷（Scratch Defect）

### 定义
划伤是 BOPET 薄膜表面沿纵向（MD方向）或横向（TD方向）的线性机械损伤痕迹，由薄膜与辊面、导辊或固定部件之间的相对滑动或硬物接触造成。

### 划伤分类
| 类型 | 特征 | 可能原因 |
|------|------|---------|
| 纵向连续划伤 | 沿 MD 方向的直线划痕 | 辊面有异物/损伤、辊面温度异常导致膜粘辊 |
| 纵向断续划伤 | 沿 MD 方向的断续划痕 | 张力波动导致膜与辊周期性打滑 |
| 横向划伤 | 垂直于 MD 方向的划痕 | 横向拉伸段（TD）问题，不在本次分析范围 |

### 划伤的工艺参数关联机理

**1. 辊温异常 → 划伤**
- 辊温过高（>正常+3°C）：PET 膜表面软化，粘辊风险增加，剥离时产生划伤
- 辊温过低（<正常-5°C）：PET 膜表面硬化，刚性增加，与辊面摩擦增大
- 辊温突变（>2°C/10min）：局部热膨胀导致辊面不平或膜张力变化

**2. 扭矩异常 → 划伤**
- 扭矩突增（>均值+3σ）：可能表明辊面摩擦增大（异物、粘辊）
- 扭矩波动增大（标准差增加）：张力不稳定，膜与辊之间打滑
- 负扭矩区正向偏移（如 W1C86 正常-75~-50，偏移至-30~0）：被动辊变主动，可能粘辊
- 相邻辊扭矩差异增大：速度不匹配，膜受剪切力

**3. 挤出压力异常 → 划伤**
- 压力波动（F_PS002 std > 0.5 MPa）：熔体流出不均匀 → 膜厚不均 → 局部张力集中
- 过滤器压差增大（F_PS002 - F_PS003 > 7.5 MPa）：过滤不畅 → 熔体含杂质 → 膜面粗糙 → 划伤加剧
- 计量泵速度波动（W1C01 std > 0.3）：挤出量不稳定 → 膜厚波动

**4. 线速度变化 → 划伤**
- 线速度突降（W1C40 < 10）：薄膜松弛，缠绕在辊上
- 线速度波动（std > 2）：张力不稳定

### 诊断关键指标

| 指标 | 计算 | 正常范围 | 异常阈值 | 关联缺陷 |
|------|------|---------|---------|---------|
| 辊温标准差 | std(TH001-TH018) | < 0.3°C | > 0.8°C | 温度不均→划伤 |
| 扭矩波动指数 | std(扭矩)/mean(abs(扭矩)) | < 0.15 | > 0.30 | 张力不稳→划伤 |
| 相邻辊扭矩差 | abs(T_i - T_{i+1}) | < 10 | > 20 | 速度不匹配→划伤 |
| 挤出压力CV | std(F_PS)/mean(F_PS) | < 0.03 | > 0.08 | 膜厚不均→划伤 |
| 过滤器压差 | F_PS002 - F_PS003 | 6.5-7.5 MPa | > 8.0 MPa | 熔体质量差→划伤 |

## 控制逻辑

- 挤出机螺杆转速由产量目标设定，计量泵控制精确挤出量
- MD 拉伸辊温度由各区独立 PID 控制
- 拉伸比由快辊/慢辊速比决定（通常 3.2-3.8x）
- 扭矩是被动变量，反映薄膜张力状态
- 线速度（W1C40）是全线速度基准

## 质量检测

- CCD 相机在线检测薄膜表面缺陷
- 每个轴号（卷号）记录各类缺陷数量
- 缺陷按类型分类：划伤、蚊虫、异物斑、粉尘、低聚物等
- 统计指标：个/100米（每100米薄膜的缺陷数）
