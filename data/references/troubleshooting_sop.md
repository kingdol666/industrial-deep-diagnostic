# 烘箱温度异常排查 SOP

## 适用范围
PVA光学膜涂布线烘箱段温度异常

## 排查步骤

### 第一步: 确认温度偏差
- 检查烘箱各段温度是否超出设定值 ±3°C
- 确认偏差持续时间 (>5分钟需介入)

### 第二步: 检查PID控制器
- 查看PID输出是否正常跟踪
- 检查是否有积分饱和现象
- 确认加热器功率响应是否正常

### 第三步: 检查加热系统
- 检查加热器工作状态
- 确认固态继电器(SSR)是否正常
- 检查热电偶读数是否准确

### 第四步: 评估对产品质量的影响
- 检查透光率是否低于90%
- 检查雾度是否高于2%
- 检查涂布厚度是否偏离目标值

## 关键因果链

```
烘箱温度过高 → 表面干燥过快 → 结皮效应 →
  → 内部溶剂无法逸出 → 微气泡/孔隙 →
  → 透光率下降 + 雾度上升
```

## 紧急处理
1. 降低烘箱温度设定值 5-10°C
2. 降低涂布速度 10-20%
3. 检查批次质量，标记受影响产品
