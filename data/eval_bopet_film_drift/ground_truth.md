# Ground Truth — BOPET Film MD Heater Degradation + Die Gap Asymmetry

## 场景信息
- **过程类型**: BOPET薄膜双向拉伸（连续过程）
- **数据规模**: 2500行 × 1min = ~42小时
- **分组列**: `product_grade`, `reel_id`, `batch_id`

## 真实根因（双根因叠加）
1. **MD_TH003加热带氧化退化**（主因~90%贡献）: 电阻↑→加热功率↓→实际温度从95°C降至~86°C→低于PET最优结晶温度→MD厚度波动+熔斑率上升
2. **模口螺栓热不对称**（次因~10%贡献）: 右侧模口冷却→热膨胀不一致→die_gap_pos4/5间隙增大4-8μm→TD厚度梯度→收卷张力不均

## 物理机制
```
机制1 (MD温度):
MD_TH003加热带氧化 → T_actual 95→86°C (低于Tg+40°C最优区间)
  → 拉伸比不一致 → 厚度波动 ±2.5μm
  → 熔斑率 0.15→~3.5 | 膜点计数 2→~8

机制2 (模口间隙):
右侧模口螺栓冷却 → 热膨胀不对称
  → die_gap_pos5 870→~880μm (左侧850μm恒定)
  → TD厚度带宽 0.3→~3.5μm
  → 收卷张力波动 → 划痕率 0.05→~1.5
```

## 关键判别特征
1. **MD_TH003是唯一单调漂移的MD温区** — 其他11个温区均稳定±1°C
2. **筛网更换(t=1200)**: filter_pressure重置，但film_thickness**不重置** → 排除"筛网堵塞"假设
3. **die_gap_pos4/5与td_thickness_band正相关** — 模口间隙→TD厚度的物理链
4. **melt_spot_rate与md_th003呈非线性负相关** — 温度<93°C后加速恶化（物理阈值）

2. **材料IV批次**: 0.59-0.65, 低IV时拉伸对温度更敏感（交互效应，非独立因）
3. **断纸(t=1900)**: 10分钟扰动→异常检测应识别为孤立事件
4. **产品等级**: Grade-A/B/C有基线差异但等级内趋势一致
