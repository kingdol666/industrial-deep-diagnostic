# 真实根因说明（禁止泄露给诊断分析系统）

## 场景：造纸机流浆箱上浆系统 — PCC/纤维沉积

### 数据文件说明

| 文件 | 内容 | 采样 |
|------|------|------|
| `DCS_export_20251001_20251229.csv` | DCS过程参数（64,800行，2分钟间隔，90天） | 每2分钟 |
| `QCS_quality_export_20251001_20251229.csv` | QCS质量数据（~3,240行，每卷采样） | 每30-50分钟 |
| `operator_event_log_2025Q4.csv` | 事件日志（纸种切换、断纸、维护） | 事件驱动 |
| `merged_dcs_qcs_20251001_20251229.csv` | DCS与QCS的时序对齐合并 | 按QCS时间点 |

### 真实根因

**PCC填料与细小纤维在上浆管道弯头和管束中渐进沉积 → 有效流通管径缩小 ~14% → 流浆箱横幅喷浆不均匀**

### 物理机制

```
管壁沉积（不可直接观测）→ 有效管径 D↓ → 压降 ΔP ∝ 1/D⁵ ↑
    ↓
流浆箱管束流量分布不均 → CD定量波动 ↑ → 成形指数 ↓
    ↓
纸张匀度劣化 + 横幅不均 → 缺陷等级 B/C
    ↓
（操作工补偿：增加助留剂 → 更多沉积 → 正反馈）
```

### 关键特征

- **压降从 ~15 kPa 升至峰值 ~32 kPa**，物理合理（∝ 1/D⁵）
- Day 55-62 有酸洗事件（`BOIL_OUT`），压降短暂下降约 50% 后重新上升
- 缺陷率：A=43%, B=31%, C=26%

### 干扰信号（分析者需识别并排除）

1. **真空泵 #2 皮带打滑（Day 22-26）** — 与沉积完全无关的独立事件
   - `vacuum_pump2_kPa` 从 48→38 kPa
   - 操作日志中有 `'Vacuum pump #2 belt replaced'`
   - **如果分析系统将此关联到质量缺陷 → 误诊**
2. **浆料温度季节性漂移（全时段）** — 同时与 defect 相关但不是因果关系
   - `stock_temp_C` 从 42→48°C（入夏自然升温）
   - 温度影响脱水速度，产生与沉积相似的缺陷趋势
   - **经典混淆变量 — 分析系统必须做偏相关/去趋势来排除**
3. **压力变送器校准（Day 40）** — `headbox_pressure_kPa` 阶跃 +0.35 kPa
   - 事件日志有记录
   - 如果不做变化点检测，会把标定漂移误认为过程真实变化

### 分析者预期路径

1. `pressure_drop_kPa` 与所有质量指标强相关（沉积的直接效应）
2. 按纸种 GSM80/100/120 分层后，压降-CD波动关系应 UNIVERSAL
3. 浆料温度的季节性趋势应通过去趋势分析（detrending）排除
4. 真空泵事件与质量无关 → 可通过时序对齐排除
5. Day 55-62 的酸洗事件提供了「自然实验」—— 清洗后质量部分恢复，确认根本原因
