## 1. 执行摘要 / Executive Summary

本报告对 CNC 铣削加工仿真数据集（1500个零件，3种材料，19把刀具，30天生产周期）进行全面的工业诊断分析。**核心发现：主轴轴承渐进式退化是导致产品质量急剧恶化的主导根因。** 主轴温度阈值为 50°C，振动阈值为 2.0 mm/s——超过任一阈值，缺陷率接近 100%。过程能力严重不足（Cp ≈ 0.5-0.6），需立即采取纠正措施。

## 2. 推理概述 / Reasoning Overview

### 2.1 数据特征 / Data Characterization
- 1500条记录，21列，无缺失值，时间顺序排列
- 3种材料：AL7075（Aluminum）、SS304（Stainless Steel）、TI6AL4V（Titanium Alloy）
- 质量等级：A（合格40.4%）、B（次品26.3%）、C（废品33.3%）
- 过程参数包括：主轴转速、进给率、切削深度、冷却液参数、主轴温度/振动、切削力等

### 2.2 统计发现 / Statistical Discovery
- **主效应量惊人**：主轴温度 Cohen's d=7.98，主轴振动 d=6.39，冷却液温度 d=4.11（A级 vs C级）
- **时间退化显著**：前半段缺陷率 26.1% → 后半段 93.1%
- **过程能力严重不足**：表面粗糙度 Cp=0.608，Cpk=0.522；尺寸偏差 Cp=0.522，Cpk=0.404
- **清晰阈值**：主轴温度 > 50°C → 99.2% 缺陷率；振动 > 2.0mm/s → 100% 缺陷率
- **预测能力**：主轴温度 > 55°C，后续10个零件 100% 不合格

### 2.3 验证过滤 / Validation Filter
- 主轴转速（p=0.142）和进给率（p=0.172）ANOVA 不显著 → H4排除
- 环境温度（p=0.880）和切削深度（p=0.113）不显著 → 非相关因素
- 所有其他过程参数在缺陷等级间差异显著（p<0.001）

### 2.4 假设演变 / Hypothesis Evolution
初始5个竞争假设 → 排除1个（H4过程参数漂移）→ 部分排除2个（H2纯刀具磨损，H3纯冷却液退化）→ 最终存活2个竞争假设（H1主轴轴承退化为主，H5轴承+刀具联合效应为次）

### 2.5 关键观测与推断 / Key Observations vs Inferences
| 观测 | 推断 |
|------|------|
| Q5时间块刀具年龄与Q1相同（37.7），但缺陷率100% vs 26% | 退化非刀具驱动，而是独立于刀具的机制 |
| 同一刀具年龄组内，A级零件主轴温度恒~38°C，C级恒~68°C | 主轴状态主导质量，刀具年龄贡献可忽略 |
| 主轴温度 > 50°C预测后续99.3%缺陷率 | 主轴状态具有时间因果先序性 |
| 冷却液温度升高但滞后于主轴温度 | 冷却液温度升高是结果而非原因 |

### 2.6 排除的假设 / What We Ruled Out
- **H4（工艺参数漂移）**：完全排除，主轴转速和进给率在缺陷等级间无显著差异
- **H2（纯刀具磨损）**：基本排除，主轴条件在控制刀具年龄后仍完全主导质量分类
- **H3（纯冷却液系统退化）**：基本排除，冷却液温度升高是主轴热负荷增加的结果

### 2.7 不确定性边界 / Uncertainty Boundaries
- 缺少主轴轴承直接检测数据（加速度包络分析、声发射监测）
- 缺少主轴维修干预的对照实验
- 无法完全排除H5（联合效应），因为主轴退化会掩盖刀具贡献

## 11. 可视化证据 — 逐图分析

### 11.1 质量指标时间趋势
![质量指标趋势](03_figures/01_quality_trends.png)

**图表展示内容**: 表面粗糙度、颤痕数、尺寸偏差、硬度在1500个零件生产序列上的散点图，颜色按缺陷等级标记。

**可视化发现 [OBSERVATION]**: 前半段（序列号0-750）零件主要集中在A级（绿色），表面粗糙度 <1.0µm，尺寸偏差 <0.02mm。序列号750之后急剧恶化，C级（红色）占主导，粗糙度达到 2.0-2.9µm，尺寸偏差 >0.08mm。硬度（HV）按材料分为三个明显水平带，不随时间显著变化。

**诊断含义**: 时间相关退化机制是系统中存在的最强信号。硬度分层仅反映材料差异，进一步确认硬度非退化标志物。

### 11.2 工艺参数-质量指标相关性热图
![相关性热图](03_figures/02_correlation_heatmap.png)

**图表展示内容**: 10个工艺参数与4个质量指标之间的Pearson相关系数矩阵。

**可视化发现 [OBSERVATION]**: 主轴振动与粗糙度的相关性 r=0.993（近乎完美线性）。主轴温度与尺寸偏差 r=0.991。冷却液温度与粗糙度 r=0.875。刀具年龄与质量的相关性仅 r=0.10-0.15。主轴转速和进给率与质量指标几乎无关（r≈-0.04）。

**诊断含义**: 主轴热态和动态参数是质量退化的主导解释变量。刀具年龄影响很小。切削参数（转速、进给率）不是退化因素。

### 11.3 关键工艺参数按缺陷等级分布
![参数按等级分布](03_figures/03_params_by_grade.png)

**图表展示内容**: 主轴振动、主轴温度、冷却液温度、切削力、刀具年龄、主轴转速在A/B/C三级间的箱线图。

**可视化发现 [OBSERVATION]**: 主轴振动和温度在三级间完全分离（A级 ~0.6mm/s/38°C，B级 ~1.3mm/s/45°C，C级 ~4.8mm/s/68°C）。冷却液温度部分重叠。刀具年龄的B级中位数高于C级——非单调关系。主轴转速三级分布几乎完全相同。

**诊断含义**: 主轴振动和温度是质量分类的完美判别器。刀具年龄与缺陷等级非单调，说明最差零件并不总是发生在刀具寿命末期。

### 11.4 刀具年龄对缺陷率的影响
![刀具年龄效应](03_figures/04_tool_age_effect.png)

**图表展示内容**: 柱状图显示缺陷率随刀具年龄增加而从41.6%上升到86.4%，折线叠加表面粗糙度均值。

**可视化发现 [OBSERVATION]**: 缺陷率随刀具年龄单调递增是真实的。但粗糙度均值增幅较温和（1.26→1.56µm）。70-80组的粗糙度（1.56µm）仍远低于C级平均粗糙度（2.33µm）。

**诊断含义**: 刀具年龄的效应真实但微小。缺陷率增加的部分原因可能是主轴状态退化恰好与时间相关——即刀具年龄可能是时间混淆变量而非因果关系。

### 11.5 主轴温度与振动趋势
![主轴温度振动趋势](03_figures/05_spindle_temp_vib_trend.png)

**图表展示内容**: 主轴温度（红色）和振动（蓝色）在1500件序列上的散点图，颜色标记缺陷等级。

**可视化发现 [OBSERVATION]**: 序列0-600：主轴温度稳定在35-42°C，振动<1.0mm/s，几乎全A级（绿色）。序列600-900：过渡区，部分B级出现，温度升至45-55°C。序列900-1500：温度陡升至65-75°C，振动升至4-6mm/s，全C级（红色）。三个明确的状态区域：健康区、过渡区、失效区。

**诊断含义**: 主轴状态呈现三级跃迁模式——与轴承磨损演化的三个阶段（稳定磨合→渐进磨损→急剧退化）高度一致。这是轴承退化经典模式。

## 12. 诊断结果 / Diagnostic Findings

### 12.1 证据排除的假设 / Evidence-Eliminated Hypotheses

| 假设 | 排除原因 | 排除置信度 |
|------|----------|:----------:|
| H4: 工艺参数漂移 | 主轴转速 p=0.142、进给率 p=0.172——ANOVA不显著，d≈-0.1 | 95% |
| H2: 纯刀具磨损 | 控制刀具年龄后，主轴条件完全主导质量分类；Q5与Q1刀具年龄相同但缺陷率100% vs 26% | 60% |
| H3: 纯冷却液退化 | 冷却液温度升高是主轴热负荷增加的结果，非独立成因 | 65% |

### 12.2 存活的假设及推理 / Surviving Hypotheses with Reasoning

| 假设 | 权重 | 置信度 | 核心证据 |
|------|:----:|:------:|----------|
| **H1: 主轴轴承退化（主导）** | 0.70 | 70 | 主轴温度 d=7.98, 振动 d=6.39；控制刀具年龄后仍完全主导；温度>55°C预测100%缺陷率；三级跃迁模式符合轴承退化经典曲线 |
| **H5: 轴承退化+刀具管理（联合）** | 0.25 | 25 | 刀具年龄有独立但微弱的效应；在轴承健康的条件下（如早期生产），刀具年龄对质量有一定影响 |

### 12.3 因果链模型 / Causal Chain Models

**H1 物理因果链**:

```
主轴轴承渐进式磨损
  → 轴承滚道/滚动体表面退化
  → 摩擦增大 → 主轴温度升高 (从38°C升至>70°C)
  → 热变形/热伸长 → 主轴-刀具系统定位精度丧失 → 尺寸偏差增大
  → 动不平衡加剧 → 主轴振动增大 (从0.6mm/s升至>5mm/s)
  → 切削过程不稳定/颤振 → 表面粗糙度恶化 & 颤痕产生
  → 产品缺陷等级从A级降级为B/C级
```

**每个环节均有数据支持**，以主轴温度和振动为可观测代理指标。

## 13. 根因分析 — 综合 / Root Cause Analysis — Synthesis

**主导根因：主轴轴承渐进式退化。**

置信度：65/100（按v6.0 COMPETING_SET协议上限）。

关键证据链：
1. 时间单调退化（26.1% → 93.1%缺陷率）——时间因果先序性 ✓
2. 主轴温度与振动效应量巨大（d=7.98, 6.39）——统计显著性 ✓
3. 清晰物理机制（轴承磨损→发热→振动→精度丧失）——物理机制 ✓
4. 控制刀具年龄后主轴条件仍完全主导——无矛盾证据 ✓

辅助因素：刀具管理（25%权重）——刀具使用至寿命末期会加剧但不能独立解释退化。

## 15. 竞争假设披露 / Competing Hypotheses Disclosure (v6.0)

**诊断类型：COMPETING_SET**

存活假设：
- **H1（主轴轴承退化）**：权重0.70。理由见§13。数据支持非常强，但缺少主轴轴承直接检测证据。
- **H5（轴承+刀具联合效应）**：权重0.25。在轴承健康前提下，刀具可能有一定贡献。但当前数据中主轴条件完全主导。

**区分条件**：若在生产线中段更换主轴轴承并观察质量是否立即恢复，可确认H1。若更换主轴后还需要更换刀具才能恢复质量，则为H5。

**区分所需数据**：主轴加速度包络分析、声发射监控数据；或维修干预实验。

## 17. 局限性与不确定性分析

### 17.1 偶然不确定性
- 测量噪声在传感器读数中不可避免
- 每个缺陷等级内存在自然变异（如A级粗糙度 0.536-1.0µm）

### 17.2 认知不确定性
- **缺少主轴轴承直接检测**：无法区分"轴承磨损"和"主轴其他部件退化"
- **缺少对照实验**：无法验证"更换主轴→质量恢复"的因果反馈
- **TH6 材料样本较少（375件）**，子组分析可能不稳定

### 17.3 什么会改变我们的结论
- 若更换主轴轴承后质量未恢复 → H1被排除，需重新诊断
- 若发现冷却液温度独立于主轴温度上升 → H3可能复活
- 若发现机床地基松动或夹具退化 → 添加新的竞争假设

### 17.4 推理链弱点
- 推理链R7（冷却液温度是结果而非原因）基于物理推理而非统计分析，证据等级较低
