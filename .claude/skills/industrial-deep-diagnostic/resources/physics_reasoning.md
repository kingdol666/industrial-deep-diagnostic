# 物理推理方法学指南 (v8.0)

本文件是 Interpreter Agent 的物理推理**方法学参考**。提供推理框架和通用物理检查方法，适配任何工业场景。

**设计原则**：
- 不是预写的定律库，而是"如何做物理推理"的方法学指南
- 每个工业场景的物理定律不同，但推理方法相同
- 附带 BOPET 和 CNC 领域的具体例子作为参考

## 1. 通用物理推理框架

### 1.1 Q1-Q5 推理链（核心方法）

对每个参数-缺陷关联，必须回答 5 个问题：

| 问题 | 回答方法 | 信息来源 |
|------|---------|---------|
| Q1: 参数的物理量是什么？ | 从列名、单位、数值范围推断 | 列名+数据 |
| Q2: 参数变化→哪个物理变量改变？量级？ | 根据领域知识推断 | 用户描述+通用物理 |
| Q3: 物理变量→哪个过程状态改变？ | 根据工艺流程推断 | 用户描述+领域知识 |
| Q4: 过程状态→能否产生观察到的缺陷？量级匹配？ | 对比原因量级 vs 效果量级 | 数据+物理公式 |
| Q5: 有没有物理定律可以定量计算？ | 查找适用的物理/化学定律 | 通用物理知识 |

### 1.2 物理可行性检查（适用于所有工业场景）

| 检查 | 方法 | 不通过标准 |
|------|------|----------|
| 量级匹配 | 计算原因的效应量级 vs 缺陷量级 | 差 >100× → 排除 |
| 温度-反应检查 | Arrhenius: k(T1)/k(T2) = exp[Ea/R × (1/T1-1/T2)] | 远离反应温度 → 排除 |
| 时间尺度 | τ_process vs τ_required | τ << τ_required → 排除 |
| 守恒律 | 能量/质量/动量守恒 | 违反 → 排除 |
| 因果时序 | 参数变化必须先于缺陷 | 倒置 → 排除 |

## 2. 通用物理定律速查（按效应类型）

以下列出工业诊断中常见的物理定律，作为推理参考。具体数值根据实际工艺场景调整。

### 2.1 温度相关效应

| 定律 | 公式 | 适用场景 |
|------|------|---------|
| 热膨胀 | ΔL = α × L × ΔT | 温度→尺寸变化（CNC主轴伸长、管道膨胀等） |
| Arrhenius 动力学 | k(T) = A × exp(-Ea/RT) | 温度→化学反应速率（降解、固化等） |
| Newton 冷却 | dT/dt = -hA(T-T_env)/(mCp) | 冷却速率计算 |
| 粘度-温度 | η(T) = η₀ × exp(Ea_visc/RT) | 温度→流体粘度变化 |

**量级估算模板**:
```
热膨胀: α_steel ≈ 12×10⁻⁶/°C, α_aluminum ≈ 23×10⁻⁶/°C
  → L=300mm, ΔT=35°C → ΔL = 12e-6 × 300 × 35 = 0.126mm

Arrhenius: k(T1)/k(T2) = exp[Ea/R × (1/T1 - 1/T2)]
  → Ea=170kJ/mol, T1=80°C(353K), T2=280°C(553K)
  → ratio = exp[170000/8.314 × (1/353 - 1/553)] = exp[16.6] ≈ 10^7.2
  → 低温段反应速率是高温段的 10^(-7.2) → 物理排除
```

### 2.2 力学相关效应

| 定律 | 公式 | 适用场景 |
|------|------|---------|
| Hooke 定律 | σ = E × ε | 应力-应变（弹性变形） |
| Taylor 切削方程 | VTⁿ = C | 切削速度-刀具寿命 |
| 颤振稳定性 | Tobias 理论 | 切削参数-颤振边界 |
| 疲劳寿命 | Basquin: σₐ = σ'f × (2Nf)^b | 循环载荷-寿命 |

### 2.3 流体/传质相关效应

| 定律 | 公式 | 适用场景 |
|------|------|---------|
| Henry 定律 | C = kH × P | 气体溶解度-压力 |
| Fick 扩散 | J = -D × dC/dx | 扩散通量 |
| Stokes-Einstein | D = kT/(6πμr) | 扩散系数-温度 |
| Darcy 定律 | Q = -kA×ΔP/(μL) | 过滤器/多孔介质流动 |

### 2.4 电磁相关效应

| 定律 | 公式 | 适用场景 |
|------|------|---------|
| Ohm 定律 | V = IR | 电流-电压关系 |
| 感应加热 | P ∝ B² × f² | 磁场-加热功率 |

## 3. Simpson's Paradox 检测方法

**这是工业诊断中最常见的统计陷阱。** 当数据中存在分组变量（材料、产品、设备等）时，整体相关可能被分组结构扭曲。

### 3.1 识别方法

```
1. 识别分组变量：材料、产品型号、设备编号、班次等
2. 在每个组内计算参数-缺陷相关
3. 比较：整体相关方向 vs 组内相关方向
4. 方向反转 → Simpson's Paradox
```

### 3.2 CNC实例（教科书级Simpson）

```
整体: feed_rate vs hardness → Spearman = -0.88（极强负相关）
  似乎结论：进给率越低→硬度越高

分层后:
  AL7075层内:  r = -0.05  → 无相关
  SS304层内:   r = +0.00  → 无相关
  TI6AL4V层内: r = -0.02  → 无相关

真相: TI6AL4V最硬(~350HV)，使用最低进给率(~1800mm/min)
      AL7075最软(~170HV)，使用最高进给率(~2900mm/min)
      → 材料差异驱动了整体相关，进给率不影响硬度
```

### 3.3 判定规则

| 整体r | 组内r | 判定 |
|-------|------|------|
| 强，方向A | 组内一致同方向A | 层内稳健，可继续推理 |
| 强，方向A | 组内方向反转或崩溃到零 | Simpson's Paradox，整体相关无效 |
| 弱 | 组内出现强相关 | 可能被整体稀释，需逐组分析 |

## 4. 领域推理参考案例

以下案例展示如何将通用推理方法应用到不同工业场景。

### 4.1 CNC 加工

```
场景: 端铣加工，主轴振动→表面粗糙度
物理推理链:
  Q1: spindle_vibration_mm/s = 主轴振动速度
  Q2: 振动增加 → 刀具径向跳动增加（正比）
  Q3: 跳动增加 → 每转切削深度不均 → 表面波纹
  Q4: 振动从0.5→4.0mm/s(+700%), 粗糙度0.7→2.5μm(+260%) → 可行
  Q5: Taylor切削方程 — 振动导致刀痕加深

量级匹配: 原因变化700% vs 效果变化260% → 可行（非线性衰减）
关键检查: 三种材料层内均>0.95 → Simpson排除
```

### 4.2 换热器

```
场景: 冷却效率下降
物理推理链:
  Q1: outlet_temp = 冷却液出口温度
  Q2: 温度升高 → 传热效率可能下降
  Q3: 结垢增加 → 热阻增加 → 传热系数下降
  Q4: ΔQ/Q = Δ(UA)/(UA) → 与观察到的温升匹配？
  Q5: Newton冷却定律 Q = UA × ΔT_lm

量级匹配: 污垢热阻 Rf 增加导致 U 下降 → ΔT上升
```

### 4.3 BOPET 薄膜（完整领域案例）

详见 `legacy/bopet_physics_reference.md`（如存在）。核心物理定律：

| 定律 | 在BOPET中的应用 |
|------|---------------|
| Arrhenius | MD段(~80°C)不可能发生化学降解，挤出段(>200°C)才是反应温度 |
| Henry 定律 | 过滤器压差→气体溶解度→气泡数量 |
| Fick/Stokes-Einstein | 低聚物在PET中的扩散极慢(~10⁻¹³ m²/s at 80°C) |
| 聚合物粘弹性 | Tg以上温度→屈服应力变化→拉伸力学响应 |

## 5. 量级匹配快速检验表

| 原因量级 | 效果量级 | 判定 |
|---------|---------|------|
| < 0.1% | > 10% | 排除（量级差 >100×） |
| 0.1-1% | > 10% | 极弱，需极强统计证据 |
| 1-10% | > 10% | 弱机制，需更多证据 |
| 10-200% | > 10% | 同量级，物理可行 |
| > 200% | < 10% | 可能（非线性衰减，部分原因转化为效果） |
