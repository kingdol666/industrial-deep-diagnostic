下部打算：高优先级优化挤出段 + 补齐关键数据缺口

高优先级

挤出段优化 · 需补充数据验证

降低挤出机各区温度

在充分熔融前提下降低设定温度。Arrhenius: 每降10°C降解速率减半 → 减少全缺陷类型

缩短熔体停留时间

提高螺杆转速 / 降低模头阻力。降解产物 ∝ 温度×时间 → 减少低聚物和凝胶

加强真空排气

检查真空泵效率，目标 <50 mbar → 有效脱除挥发性降解产物 → 减少气泡

建立定期清理计划

清理螺杆/模头残留物 → 解决melt_spots持续上升 (时间趋势r=0.379)

中优先级

MD段监测优化 · 低实施难度

建立型号内SPC控制图

消除Simpson悖论误判 — 聚合数据会掩盖型号内变异。按型号分别监控

监控扭矩波动趋势

W1C83@PV1_std — 扭矩波动是黏度变化的代理信号，可用于早期预警降解程度变化

减少型号切换频率

型号切换时的过渡状态缺陷率通常更高 → 减少过渡批次浪费

严格控制原料含水率

目标 <50 ppm → 水分超标会显著加速IV下降+增加水解气泡

补充数据

最迫切需要 · 按优先级排列

1. 挤出机各区温度

MG-T1~T6, SG-T1~T4 — 直接验证热降解假设，可将置信度从72提升至85+

2. 螺杆转速和熔体泵转速

估算熔体停留时间 — 量化降解程度的关键参数

3. 原料PET IV和含水率

区分原料因素与工艺因素 — 明确各因素贡献比例

4. 1-3个月长周期数据

提高统计效力 · 验证长期退化趋势 · Granger因果需要更长时序列
